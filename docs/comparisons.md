# Comparisons

<div class="bench-header" markdown>
<div markdown>

How `decimal-scaled` compares on **speed** against other Rust decimal and numeric
crates. Each function below has a table and a grouped bar chart: the median time
of every library at each storage width, measured at a *typical* scale for that
width — the quarter-of-maximum point, since most code uses far fewer fractional
digits than a tier can hold. The multiplier beside each peer is its time relative
to `decimal-scaled`; a library with **no bar at a width** has no equivalent type
there. Generated from CI (`lib-cmp-perf`); refreshed on each release PR.

</div>
