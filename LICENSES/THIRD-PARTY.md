Third-party software incorporated into this crate.

This crate incorporates no third-party source code, in either vendored
or adapted form. All algorithm implementations are the crate's own,
written from primary sources (academic papers and standards). For the
*idea* attributions — the published algorithms each kernel implements,
with citations — see `ALGORITHMS.md`.

Should any upstream code ever be vendored or adapted in future, each
instance will be listed here with its upstream identification, the
pinned commit / version, the license terms in full, and a pointer to
where the material lives in this tree.
